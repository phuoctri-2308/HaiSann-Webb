# Web bán hàng demo (responsive mobile)

## Cấu trúc
- `sanpham.html`: Trang sản phẩm + giao diện
- `styles.scss`: CSS viết theo kiểu SCSS (biến, nesting có thể mở rộng)
- `styles.css`: File CSS dùng trực tiếp trong trình duyệt
- `app.js`: JavaScript xử lý tìm kiếm, lọc danh mục, giỏ hàng (localStorage)

## Chạy nhanh
Chỉ cần mở trực tiếp `sanpham.html` trong trình duyệt.

> Nếu bạn muốn build SCSS -> CSS, cài SASS CLI:
> - `npm i -g sass` (nếu có node)
> - build: `sass styles.scss styles.css --no-source-map`
